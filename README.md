## SpaceArcade
Made some simple shooter game from scratch for studying purposes.
# Controls:
+ W,↑ -fly forward;
+ S,↓ -fly backward;
+ A,← -fly left;
+ D,→ -fly right;
+ Space - shot lazer bolts;
+ R - restart;
